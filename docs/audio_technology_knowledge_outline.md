# 音频技术科普知识大纲

本文档用于为后续网页内容建设准备音频技术科普知识体系。内容覆盖音频基础、硬件、软件、传统算法、AI 算法以及典型应用场景，后续可拆分为网页栏目、专题文章、知识卡片或交互式知识图谱。

## 1. 音频技术总览

### 1.1 什么是声音

- 声音的物理本质：振动、声波、介质传播。
- 频率、振幅、相位、波长。
- 人耳听觉范围与听感差异。
- 声音从产生、传播到被感知的基本过程。

### 1.2 模拟音频与数字音频

- 模拟信号与数字信号的区别。
- 采样、量化、编码。
- 采样率、位深、动态范围、信噪比。
- PCM、DSD、压缩音频的基本概念。

### 1.3 音频系统的基本链路

- 声音采集：麦克风。
- 信号转换：ADC / DAC。
- 处理与传输：DSP、Codec、蓝牙、USB、网络。
- 播放输出：功放、扬声器、耳机。
